# NIAGARA (나이아가라) 🌊

> **"폭포처럼 쏟아지는 정보, 간단하게"**  
> 시니어를 위한 음성+터치 하이브리드 플랫폼

---

## 🎯 프로젝트 개요

**NIAGARA**는 50대~70+ 시니어를 위한 **Universal Design 기반 웹 플랫폼**입니다.  
"터치 우선, 음성 보조" 하이브리드 모델로 디지털 접근성을 높이고, 전문적이면서도 쉬운 UX를 제공합니다.

### 핵심 가치
- ✅ **큰 글씨 (18pt~26pt)** - 노안도 편한 가독성
- ✅ **고대비 색상 (#1A237E)** - 명확한 시각적 구분
- ✅ **음성 명령** - "오늘 날씨 알려줘" 등 자연어 처리
- ✅ **터치 최적화** - 48px 이상 큰 터치 타겟
- ✅ **쉬운 되돌리기** - 실수 두려움 해소

---

## 🚀 현재 구현된 기능 (MVP v1.0)

### ✅ 완료된 기능

#### 1. **홈 화면**
- 큰 아이콘 4개 (뉴스, 날씨, 건강, 설정)
- 글씨 크기 조절 (작게/보통/크게)
- 음성 검색 버튼 (헤더 고정)
- 오늘의 주요 뉴스 (3건 미리보기)
- 하단 네비게이션 바 (홈/뉴스/건강/내정보)

#### 2. **뉴스 페이지**
- Mock 뉴스 데이터 3건
- 카테고리별 뱃지 (사회/날씨/건강)
- "읽어주기" 버튼 (TTS 음성 출력)
- "공유하기" 버튼 (예정)

#### 3. **설정 페이지**
- 글씨 크기 조절
- 고대비 모드 토글 (예정)
- 음성 안내 ON/OFF
- 앱 정보 (버전, 문의)

#### 4. **음성 인식 (STT)**
- Web Speech API 활용
- 한국어 인식 (`ko-KR`)
- 지원 명령어:
  - "글씨 크게/작게/보통"
  - "뉴스/날씨/건강/설정/홈"
  - 음성 명령 피드백 (TTS)

#### 5. **음성 출력 (TTS)**
- 버튼 클릭 시 음성 안내
- 뉴스 읽어주기 기능
- 명령 실행 확인 음성

## 👥 사용자 가이드

### **1. 홈 화면에서 시작**
- 큰 아이콘 4개 중 하나를 터치하세요
- 또는 "음성 검색" 버튼을 눌러 말하세요

### **2. 글씨 크기 조절**
- 홈 화면 상단 "글씨 크기" 섹션
- 작게/보통/크게 버튼 클릭
- 또는 "글씨 크게 해줘" 음성 명령

### **3. 음성 명령 사용법**
1. "음성 검색" 버튼 클릭
2. 마이크 권한 허용 (최초 1회)
3. 명령어 말하기:
   - "오늘 뉴스 보여줘"
   - "날씨 알려줘"
   - "글씨 크게"
   - "홈으로 가줘"

### **4. 뉴스 읽기**
- 뉴스 페이지에서 "읽어주기" 버튼
- 자동으로 뉴스 제목+내용 음성 출력
