# ZZonde (쫀드기) 🍬

> **"인생이 쫀득해진다"**  
> 시니어를 위한 음성+터치 하이브리드 플랫폼 + **AI 동반자 '존디'**

---

## 🎯 프로젝트 개요

**ZZonde (쫀드기)** 는 한국 추억의 쫀득한 과자에서 영감을 받은 **Universal Design 기반 웹 플랫폼**입니다.  
50대~70+ 시니어를 위한 "터치 우선, 음성 보조" 하이브리드 모델로 디지털 접근성을 높이고, 레트로-힙한 감성과 쫀득한 사용자 경험을 제공합니다.

### 핵심 가치
- 🍬 **쫀득한 UX** - 끈적이듯 부드러운 인터랙션
- 🤖 **AI 동반자 '존디'** - 기억하고 공감하는 디지털 친구
- 🎨 **레트로-힙** - 추억을 현대적으로 재해석
- 🧡 **활력 넘치는 오렌지** - 식욕과 에너지를 자극하는 비브런트 컬러
- ✅ **큰 글씨 (18pt~26pt)** - 노안도 편한 가독성
- ✅ **고대비 색상** - 오렌지 & 옐로우의 명확한 시각적 구분
- ✅ **음성 명령** - "오늘 날씨 알려줘" 등 자연어 처리
- ✅ **터치 최적화** - 48px 이상 큰 터치 타겟
- ✅ **안전 모니터링** - 위급 상황 자동 감지 및 알림

---

## 🤖 NEW! AI 동반자 '존디' (Companion AI)

### 핵심 기능

#### 🧠 1. Memory System (기억 시스템)
- **대화 내역 저장**: 최대 100개의 대화를 LocalStorage에 저장
- **사용자 컨텍스트**: 건강 상태, 가족 관계, 취미, 걱정거리 자동 추출
- **감정 분석**: 슬픔, 기쁨, 걱정, 화남 등 감정 자동 감지
- **스마트 검색**: 키워드 기반 관련 대화 검색
- **컨텍스트 주입**: AI 프롬프트에 과거 대화 내용 자동 포함

#### ⚠️ 2. Safety Monitor (안전 감시)
- **위급 키워드 감지**: "살려줘", "너무 아파", "도와줘" 등 자동 감지
- **감정 패턴 분석**: 지속적인 부정 감정 추적
- **즉각 대응 UI**: 위급 상황 시 119/가족 연결 버튼 팝업
- **주기적 체크**: 1분마다 사용자 안전 상태 확인
- **비상 연락처 관리**: 가족, 119 등 비상 연락처 등록

#### 🎙️ 3. Voice-First Interface
- **거대한 마이크 버튼**: 128x128px 크기의 Floating Action Button
- **실시간 STT/TTS**: Web Speech API 활용
- **대화 히스토리**: 과거 대화 자동 로드 및 표시
- **빠른 응답 버튼**: 날씨, 뉴스, 복약 시간 등 원터치

#### 🤝 4. Active Trigger (능동적 상호작용)
- **AI가 먼저 인사**: 앱 실행 시 시간대별 맞춤 인사 (하루 1회)
- **컨텍스트 follow-up**: "어제 걱정하시던 일은 좀 나아지셨나요?"
- **공감과 관심**: 과거 대화 기반 관심 표현

### **1. 홈 화면에서 시작**
- 큰 아이콘 4개 중 하나를 터치하세요
- 또는 "음성 검색" 버튼을 눌러 말하세요

- 홈 화면 상단 오렌지 배경 섹션

1. 헤더의 "음성 검색" 버튼 클릭
2. 마이크 권한 허용 (최초 1회)
3. 명령어 말하기:
   - "오늘 뉴스 보여줘"
   - "날씨 알려줘"
   - "글씨 크게"
   - "홈으로 가줘"

### **4. 뉴스 읽기**
- 뉴스 페이지에서 오렌지색 "읽어주기" 버튼
- 자동으로 뉴스 제목+내용 음성 출력
